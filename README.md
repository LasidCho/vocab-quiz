# Vocab Master - 어휘 학습 앱

유의어/반의어를 학습할 수 있는 모바일 웹 앱입니다.

## 기능

- 📱 모바일 최적화 UI
- 🔐 전화번호 기반 로그인/회원가입
- 📝 유의어/반의어/혼합 퀴즈
- 📊 주간 학습 리포트
- 📁 엑셀 파일 업로드로 단어 추가
- ⚙️ 비밀번호 변경

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Supabase (백엔드)

## 배포 방법

### Vercel 배포

1. GitHub에 이 프로젝트를 업로드
2. [Vercel](https://vercel.com)에서 GitHub 연결
3. 프로젝트 Import → 자동 배포

## 테스트 계정

- 전화번호: `01000000000`
- 비밀번호: `admin123`
